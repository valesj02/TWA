# Tire Change Reservation System

This is a simple web app for booking tire change appointments using Firebase Realtime Database and JavaScript.

## Features
- Create a new reservation
- View all existing reservations
- Update a reservation
- Delete a reservation

## Setup
1. Create a Firebase project at https://console.firebase.google.com
2. Enable **Realtime Database** in test mode.
3. Replace `your-project-id` in `app.js` with your actual Firebase project ID.
4. Open `index.html` in your browser.

## Security Note
This app uses test mode with public read/write access. Update your Firebase security rules before going live.
